I made another free Autodesk Flame Matchbox: Greg Jumbotron.

This one came from a pretty common finishing need: making footage feel like it is playing on a screen.

Not just a blur or a scanline slapped over the top, but a useful screen texture that can get me most of the way there on a Jumbotron, LED wall, monitor, phone, signage, or any other display surface.

Greg Jumbotron adds:

- LED cell / pixel pitch control
- dark gaps between LEDs
- RGB subpixel striping
- animated scanlines
- per-cell flicker
- moire texture
- a rolling refresh band
- subtle edge bloom
- preview modes for the texture, LED matte, and pixel source

The main control to animate is `Animation Phase`. Keyframe that over the shot and the screen texture comes alive.

It is a small thing, but it solves a real problem for me. I often need a screen to feel like a screen quickly, without building the same node setup from scratch every time.

The compiled `.mx` is included, along with the GLSL/XML source.

Free download:
https://github.com/woodie228/greg-jumbotron-flame

Built for Flame Matchbox / GLSL 120.
